Mon portefolio

Dans le cadre du bootcamp Le Wagon, nous avons eu, par groupe de 4, à développer deux applications web de A à Z.

Exeptions

Le premier projet avait pour thème imposé de créer un marketplace type Airbnb.

Certain ont choisi de louer des chevaux de courses, d'autres des hommes politiques corrompus.

Nous avons choisi plus modestement de louer des châteaux.

Swappy

Le deuxième projet était totalement libre.

Nous avons choisi de développer une application web mobile pour faciliter l'utilisateur à trouver des activités à proposer à ses amis.

Une fois le ou les thêmes choisis, l'application propose des activités que l'utilisateur conserve en favori avec un système de swipe. Les activités se retrouvant dans les favoris peuvent être proposé à vos contact. Un groupe de discussion est alors créé...
